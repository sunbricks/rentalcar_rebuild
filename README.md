# rentalcar Springframework Rebuild project

# 프로젝트 설명
순수 JAVA와 순수 JDBC로 구축한 렌트어때 사이트를 Springframework로 리빌딩 하는 포트폴리요옹 프로젝트입니다.

# 개발환경 & 사용기술
**백엔드**
- Java 11
- Spring Framework
- Spring MVC
- JSP
- Maven
- MyBatis
- Apache Tomcat 9
- MySQL & MySQL workbench 8.0

**프론트엔드**
- HTML5
- CSS3
- JavaScript
- jQuery
- Ajax
